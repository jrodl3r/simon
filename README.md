#Simon (Game)
Web-based version of the classic electronic memory game from the 70's & 80's. ([Wiki](http://en.wikipedia.org/wiki/Simon_%28game%29))

### Objectives
* Pure JavaScript (No jQuery)
* ES6 Classes (Traceur)
* TDD (Jasmine)
* Automation (Grunt)

### Setup
$ npm install --save-dev
$ grunt
$ server
